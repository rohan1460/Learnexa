## 🌟 Overview

**Learnexa** is a modern AI-powered study companion designed to transform how students learn. Whether you're preparing for exams, tackling complex topics, or just trying to understand something new, Learnexa brings the power of cutting-edge AI to your fingertips — all wrapped in a beautiful, intuitive interface.

Ask questions. Generate quizzes. Summarize notes. Convert formats. Listen to your content. Get code help. **All in one place.**

## 🛠️ Tech Stack

**Frontend**
- ⚡ [Next.js 15](https://nextjs.org/) — React framework with App Router
- 🎨 Custom CSS — Handcrafted, responsive designs
- 📘 [TypeScript](https://www.typescriptlang.org/) — Type-safe development

**Backend & Database**
- 🔗 Next.js API Routes — Serverless backend
- 🗄️ [Prisma ORM](https://www.prisma.io/) — Type-safe database access
- 💾 SQLite — Local-first database

**AI & Intelligence**
- 🧠 [OpenRouter](https://openrouter.ai/) — Multi-model AI gateway (GPT-4o-mini)
- ✨ [Google Gemini](https://ai.google.dev/) — Advanced AI capabilities
- ⚡ [Groq](https://groq.com/) — Lightning-fast inference
- 🔍 Vector Embeddings — Semantic search (Jina embeddings)

**Authentication**
- 🔐 [NextAuth.js](https://next-auth.js.org/) — Complete auth solution

## 📁 Project Structure

```
learnexa/
├── prisma/              # Database schema & migrations
├── public/              # Static assets
├── src/
│   ├── app/
│   │   ├── api/         # API routes (chat, quiz, tts, etc.)
│   │   ├── dashboard/   # Feature pages (chat, code, convert, etc.)
│   │   ├── login/       # Authentication pages
│   │   └── page.tsx     # Landing page

│   ├── lib/             # Utilities (AI, Prisma, vectors)
```
